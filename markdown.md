# Git Ejemplo: Repositorio de Nuvia89

Este es un repositorio de ejemplo para practicar la documentación en Markdown, el uso de ramas y la gestión de commits en Git.

## Archivos del proyecto

* **NuvixPrueba.py**: Script principal de prueba utilizado para verificar la configuración del entorno.
* **Actividad5_Equipo.ipynb**: Notebook de Jupyter con ejercicios de programación en Python.
* **markdown.md**: Documento actual que utiliza sintaxis de marcado ligero[cite: 5].

## Sintaxis utilizada en este documento

Para la elaboración de este archivo se aplicaron los siguientes elementos de Markdown :
1. **Encabezados**: Uso de `#` y `##` para jerarquía.
2. **Listas**: Elementos ordenados y desordenados para organizar la información.
3. **Tablas**: Para presentar el historial de cambios de forma legible.
4. **Negritas**: Para resaltar nombres de archivos y comandos.
